# 资料来源与版权规则

## 收录优先级

1. 官方文档、标准、政府或国际组织资料。
2. 论文、大学课程、公开研究报告。
3. 活跃维护的开源项目和知名工具文档。
4. 可复现的教程、案例和工程实践文章。

## 不收录或谨慎收录

- 无法追溯来源的转载内容。
- 只给结论、不展示过程的营销文章。
- 需要付费授权才能引用核心内容的课程或资料。
- 医疗、法律、财务等高风险领域中的非权威建议。

## 版权规则

Coding2Live 不搬运全文，不复制受版权保护的大段内容。条目应使用自己的语言做摘要、评论和实践说明，并提供原始链接。

## 评价维度

- 可信度：是否来自官方或长期维护者。
- 可复现：读者是否能跟着完成。
- 时效性：工具、API、政策是否可能过期。
- 风险提示：是否说明适用边界。
