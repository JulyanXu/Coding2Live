# 个人学习助手

## 适用人群

适合希望自学一门技能、准备考试、阅读论文或建立长期知识系统的人。

## 可解决的问题

- 把学习目标拆成阶段计划和每日任务。
- 把长资料转成提纲、术语表、练习题和复盘问题。
- 用间隔重复和项目练习巩固知识。

## Vibe Coding 工作流

1. 写清学习目标、当前水平、截止时间和可用时间。
2. 让 AI 生成学习路径和诊断题。
3. 把资料摘要为卡片、练习和项目任务。
4. 每周用 AI 对错题、笔记和项目卡点做复盘。
5. 把有效提示词和资料链接沉淀为个人知识库。

## 推荐工具

- Anki，用于长期记忆卡片。
- Notion 或 Obsidian，用于学习日志和资料索引。
- edX 等开放课程平台，用于高质量课程资料。

## 示例提示词

```text
你是我的学习教练。我想在 8 周内入门 Python 数据分析：
1. 当前水平：会基础 Excel，不会编程。
2. 每周可投入 5 小时。
3. 输出：8 周计划、每周项目、检查点、推荐练习。
4. 每个知识点都给出一个可操作的小任务。
5. 最后生成适合做 Anki 卡片的术语表。
```

## 真实/参考项目

可以从一门公开课程开始，把每节课的目标、笔记、练习和复盘统一放进一个学习看板。

## 限制与风险

AI 生成的学习路径可能忽略先修知识或过度乐观。需要通过练习结果持续校准，而不是只看计划。
