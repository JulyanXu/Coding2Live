# 家庭与生活自动化

## 适用人群

适合希望把家庭清单、预算、日程、设备和提醒统一管理的人。

## 可解决的问题

- 把家务、账单、购物、维修和家庭日程变成可追踪清单。
- 自动生成购物清单、旅行清单和家庭会议议程。
- 连接智能家居设备实现基础提醒和场景。

## Vibe Coding 工作流

1. 列出重复发生的家庭任务和触发条件。
2. 让 AI 生成清单模板、预算表和提醒规则。
3. 先用表格、日历、提醒事项和 IFTTT 验证。
4. 对智能家居需求，再评估 Home Assistant 等系统。
5. 定期删除不用的自动化，避免通知噪音。

## 推荐工具

- Home Assistant，用于本地智能家居自动化。
- IFTTT，用于简单跨服务触发。
- Google Nest 帮助文档，用于理解常见家庭设备能力。

## 示例提示词

```text
请帮我设计一个家庭任务系统：
1. 家庭成员：2 位成人、1 位小学生。
2. 目标：减少忘记账单、购物和学校事项。
3. 输出：任务分类、每周例行清单、共享日历规则、提醒策略。
4. 区分必须自动化、适合手动确认、不要自动化的任务。
5. 给出可以先用表格和日历完成的版本。
```

## 真实/参考项目

可以从共享日历和购物清单开始，稳定后再用 IFTTT 或 Home Assistant 连接设备。

## 限制与风险

家庭自动化涉及隐私和安全。门锁、摄像头、儿童信息等场景应优先使用官方安全设置，并保留人工确认。
