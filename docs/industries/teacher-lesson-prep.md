# 教师备课

## 适用人群

适合需要快速准备课程大纲、课堂练习、测验题、讲义和学习反馈材料的教师或培训师。

## 可解决的问题

- 把课程标准、教材段落和学生水平转成可执行课时设计。
- 为不同基础的学生生成分层练习。
- 把课堂活动、测验和作业组织成可复用模板。

## Vibe Coding 工作流

1. 输入教学目标、年级、课时长度和学生画像。
2. 让 AI 生成课程流程、板书结构、练习题和评价标准。
3. 用脚本或表格批量生成题库、答案和讲义。
4. 人工审查事实、难度、价值观表达和本地课程标准。
5. 把成熟模板沉淀为下一次备课的项目。

## 推荐工具

- Google AI 教育资源，用于理解 AI 在教学中的常见使用方式。
- GitHub Copilot 或通用代码助手，用于批量生成题库、表格和互动小工具。
- Moodle，用于把材料转成课程、测验和作业模块。

## 示例提示词

```text
你是一名初中科学教师。请基于“能量守恒”主题设计一节 45 分钟课程：
1. 学生基础：多数学生能理解速度和质量，但还不熟悉能量转换。
2. 输出：教学目标、课堂流程、2 个演示活动、8 道分层练习、课后作业。
3. 每个活动说明所需材料、时间和可能误区。
4. 最后列出教师必须人工核查的事实点。
```

## 真实/参考项目

可以从 Moodle 课程结构开始，把 AI 生成的练习题导入测验模块，再逐步沉淀校本题库。

## 限制与风险

AI 可能生成错误知识点或不符合本地课程标准的活动。涉及学生隐私时，不应上传可识别个人身份的信息。
