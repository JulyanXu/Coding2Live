# 案例：写作与出版

## 适用人群

需要持续产出文章、报告、电子书或课程讲义的人。

## 可解决的问题

选题发散、结构混乱、版本不可追踪、多平台格式重复处理。

## Vibe Coding 工作流

先用 AI 做选题和结构编辑，再用 Markdown 管理正文，用 Pandoc 导出网页、PDF 或电子书。

## 推荐工具

GitHub、Markdown、Pandoc、风格手册。

## 示例提示词

```text
请只做结构编辑：基于我的素材提出标题、核心论点、章节顺序、需要补充来源的位置。
```

## 真实/参考项目

建立一个书稿仓库，每章一个 Markdown 文件，发布前统一转换格式。

## 限制与风险

事实、引用和观点归属必须人工确认。
