# 案例：数据分析入门

## 适用人群

有表格数据，但不知道如何分析和表达结论的人。

## 可解决的问题

问题不清、数据质量未知、图表和结论脱节。

## Vibe Coding 工作流

让 AI 根据字段提出指标和假设，用 pandas 清洗，用 Jupyter 记录分析，再用 Streamlit 分享结果。

## 推荐工具

pandas、Jupyter、Streamlit。

## 示例提示词

```text
请根据这些字段设计分析流程：数据质量检查、关键指标、pandas 步骤、图表和误读风险。
```

## 真实/参考项目

从 CSV 做探索分析，再沉淀一个只展示核心指标的小仪表盘。

## 限制与风险

不能把相关性直接解释为因果。
