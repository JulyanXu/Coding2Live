# 案例：个人学习助手

## 适用人群

正在学习新技能，并希望把资料、练习和复盘集中管理的人。

## 可解决的问题

学习计划过大、资料过散、练习不足、复盘不连续。

## Vibe Coding 工作流

用 AI 生成学习路线和诊断题，将资料摘要为卡片和练习，每周让 AI 基于错题与项目进展提出下一步计划。

## 推荐工具

Anki、Notion/Obsidian、开放课程平台、通用 AI 助手。

## 示例提示词

```text
请根据我的目标生成 4 周学习系统：目标、每周任务、每日练习、复盘问题、Anki 卡片字段。
```

## 真实/参考项目

从一个课程开始，把每节课转成笔记、卡片、练习和项目任务。

## 限制与风险

AI 计划可能过度乐观。每周应基于真实完成情况调整。
