# 案例：医疗健康信息整理

## 适用人群

需要帮自己或家人整理就诊材料的人。

## 可解决的问题

症状记录零散、就诊时忘记关键问题、报告难理解。

## Vibe Coding 工作流

记录事实，让 AI 整理时间线和问题清单，并用权威网站核对健康知识。

## 推荐工具

表格、笔记应用、OCR、MedlinePlus、CDC。

## 示例提示词

```text
请只整理事实，不做诊断：输出症状时间线、用药记录、检查摘要和给医生的问题。
```

## 真实/参考项目

建立家庭健康日志，保留日期、症状、药物、报告和医生反馈。

## 限制与风险

紧急症状应直接联系医疗机构，不能等待 AI 分析。
