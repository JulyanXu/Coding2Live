# 案例：家庭与生活自动化

## 适用人群

希望减少家庭琐事遗漏、统一清单和提醒的人。

## 可解决的问题

购物、账单、维修、学校事项和家庭日程分散。

## Vibe Coding 工作流

把重复任务拆成触发条件和负责人，用 AI 生成清单、预算表和提醒规则，先低代码验证。

## 推荐工具

共享日历、提醒事项、IFTTT、Home Assistant。

## 示例提示词

```text
请设计家庭任务系统：分类、共享日历、提醒规则、自动化边界和每周复盘。
```

## 真实/参考项目

从共享购物清单和账单提醒开始，再接入智能家居设备。

## 限制与风险

摄像头、门锁和儿童信息必须优先考虑隐私和安全。
