# Coding2Live 百科入口

Coding2Live 的目标是把分散在官方文档、开源项目、论文、教程和真实案例中的资料整理成一套可行动的百科。它关注的不是“AI 会什么”，而是普通人在工作、学习和生活中如何把需求表达、信息整理、工具组合、原型验证和持续迭代串起来。

## 阅读方式

- 从“案例库”进入，直接查看一个场景如何落地。
- 从“行业场景”进入，理解某个职业或业务领域的边界与风险。
- 从“工作流”进入，复用需求澄清、数据处理、代码审查等通用方法。
- 从“资料来源”进入，追溯官方文档、项目和参考资料。

## 条目格式

每个条目固定包含：适用人群、可解决的问题、vibe coding 工作流、推荐工具、示例提示词、真实/参考项目、限制与风险、来源。

## 一期原则

一期只做精选，不做全网爬虫。资料不会搬运全文，只做摘要、结构化评论、实践路径和链接。
