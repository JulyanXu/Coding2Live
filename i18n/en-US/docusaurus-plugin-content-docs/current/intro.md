# Coding2Live Encyclopedia

Coding2Live organizes official docs, open-source projects, papers, tutorials, and real cases into an actionable encyclopedia. It focuses less on what AI can do in the abstract, and more on how people can combine requirement framing, information organization, toolchains, prototypes, and iteration in everyday work.

## How to Read

- Start from the case library when you want a concrete scenario.
- Start from industries when you need domain boundaries and risks.
- Start from workflows when you want reusable patterns for scoping, data work, automation, or review.
- Start from sources when you want to trace original docs and projects.

## Entry Format

Each entry includes audience, problems solved, vibe coding workflow, recommended tools, example prompts, reference projects, limitations and risks, and sources.

## Phase One Principle

Phase one is curated, not crawled. Coding2Live summarizes, evaluates, and links to sources instead of republishing full content.
