# Personal Learning Assistant

## Audience

People who want to learn a skill, prepare for an exam, read papers, or build a long-term knowledge system.

## Problems Solved

- Break goals into phases and daily tasks.
- Turn long materials into outlines, glossaries, exercises, and review questions.
- Reinforce knowledge through spaced repetition and projects.

## Vibe Coding Workflow

1. Define the goal, current level, deadline, and available time.
2. Ask AI for a learning path and diagnostic questions.
3. Convert materials into cards, exercises, and project tasks.
4. Use weekly review prompts on mistakes, notes, and blockers.
5. Save effective prompts and links in a personal knowledge base.

## Recommended Tools

- Anki for long-term memory.
- Notion or Obsidian for logs and source indexes.
- Open course platforms such as edX for high-quality materials.

## Example Prompt

```text
Act as my learning coach. I want to learn Python data analysis in 8 weeks:
1. Current level: basic Excel, no programming.
2. Time: 5 hours per week.
3. Output an 8-week plan, weekly projects, checkpoints, and practice tasks.
4. Give each concept a concrete task.
5. End with a glossary suitable for Anki cards.
```

## Reference Project

Start from one public course and turn every lesson into notes, cards, exercises, and project tasks in a learning board.

## Limitations and Risks

AI learning plans may miss prerequisites or be too optimistic. Adjust the plan using actual practice results.
