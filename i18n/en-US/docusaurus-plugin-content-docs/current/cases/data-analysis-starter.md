# Case: Data Analysis Starter

## Audience

People with spreadsheet data who do not yet know how to analyze and communicate findings.

## Problems Solved

Unclear questions, unknown data quality, and disconnected charts and conclusions.

## Vibe Coding Workflow

Ask AI to propose metrics and hypotheses from fields, clean data with pandas, record analysis in Jupyter, and share results with Streamlit.

## Recommended Tools

pandas, Jupyter, and Streamlit.

## Example Prompt

```text
Design an analysis flow from these fields: data-quality checks, key metrics, pandas steps, charts, and misinterpretation risks.
```

## Reference Project

Explore a CSV in Jupyter and turn the core metrics into a small dashboard.

## Limitations and Risks

Do not treat correlation as causation.
