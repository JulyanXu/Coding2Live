# Case: Personal Learning Assistant

## Audience

People learning a new skill who want one place for sources, exercises, and reflection.

## Problems Solved

Oversized plans, scattered materials, too little practice, and inconsistent review.

## Vibe Coding Workflow

Use AI to generate a learning path and diagnostic questions, convert materials into cards and exercises, then run weekly reviews based on mistakes and project progress.

## Recommended Tools

Anki, Notion or Obsidian, open course platforms, and a general AI assistant.

## Example Prompt

```text
Create a 4-week learning system for my goal: objectives, weekly tasks, daily practice, review questions, and Anki fields.
```

## Reference Project

Turn one course into notes, cards, exercises, and project tasks.

## Limitations and Risks

AI plans may be too optimistic. Adjust weekly based on actual completion.
