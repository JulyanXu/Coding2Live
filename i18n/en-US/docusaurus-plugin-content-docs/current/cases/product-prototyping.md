# Case: Product Prototyping

## Audience

People who need to validate a product idea and its key interactions quickly.

## Problems Solved

Vague requirements, missing states, and prototypes without acceptance checks.

## Vibe Coding Workflow

Use AI to draft user stories, pages, and states, then generate a prototype and verify paths with Playwright.

## Recommended Tools

Figma, GitHub Copilot or Codex, and Playwright.

## Example Prompt

```text
Turn this product idea into an MVP prototype spec: user stories, pages, states, acceptance paths, and non-goals.
```

## Reference Project

Build a static interactive prototype within three days, then decide whether a backend is needed.

## Limitations and Risks

Do not let prototype work expand into full product development.
