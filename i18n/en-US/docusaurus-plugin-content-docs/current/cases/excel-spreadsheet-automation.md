# Case: Excel and Spreadsheet Automation

## Audience

People who repeatedly process spreadsheets, summaries, and anomalies weekly or monthly.

## Problems Solved

Manual copy-paste, hard-to-maintain formulas, and hidden data issues.

## Vibe Coding Workflow

Ask AI for a formula-based approach first, then a pandas script, and validate each step with sample data.

## Recommended Tools

Excel, Google Sheets, pandas, and Apps Script.

## Example Prompt

```text
Generate a monthly sales reporting workflow from these fields: formulas, pivot tables, pandas script, and validation checks.
```

## Reference Project

Package a monthly sales report as raw input, anomaly list, and summary output.

## Limitations and Risks

Keep original files and sample-check financial or inventory data.
