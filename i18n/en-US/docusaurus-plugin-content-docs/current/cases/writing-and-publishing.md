# Case: Writing and Publishing

## Audience

People producing articles, reports, ebooks, or course notes on a recurring basis.

## Problems Solved

Scattered topics, weak structure, poor version tracking, and repeated format conversion.

## Vibe Coding Workflow

Use AI for topic and structural editing, manage the draft in Markdown, and export web, PDF, or ebook formats with Pandoc.

## Recommended Tools

GitHub, Markdown, Pandoc, and a style guide.

## Example Prompt

```text
Only act as a structural editor: propose titles, core argument, section order, and source gaps from my material.
```

## Reference Project

Maintain a manuscript repository with one Markdown file per chapter and a repeatable export process.

## Limitations and Risks

Facts, citations, and attribution must be confirmed manually.
