# Contributing

Coding2Live welcomes new entries, source indexes, case revisions, and toolchain verification notes. Contributions should make a workflow easier to reproduce, not merely add more links.

## Entry Requirements

- Use English in this locale and preserve original terminology when useful.
- Include at least two high-quality sources, prioritizing official docs, papers, reputable open-source projects, and reproducible tutorials.
- Do not republish full articles, long excerpts, or paid-course content.
- State limitations and risks clearly, especially in legal, medical, financial, and security-related areas.
- Example prompts should be directly reusable or easy to adapt.

## Review Priorities

Review for source quality, reproducibility, and risk clarity. Tool recommendations may change over time, but a single vendor capability should not be presented as a universal rule.
